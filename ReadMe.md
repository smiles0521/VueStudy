# Vue.js

> ​	当前最火爆的MVVM框架，以轻量、简介、高效闻名，尤其是数据驱动、组件化的核心思想，大幅度的提升开发效率。

## 学习目标

* 掌握Vue.js在实战中的运用；
* 学会使用Vue.js完整的开发移动APP；
* 学会组件化、模块化的开发方式；

## 学习内容

* Vue.js 框架介绍
* Vue-cli 脚手架  搭建基本代码框架
* Vue-router  官方插件管理路由
* Vue-resource   Ajax通信
* webpack    构建工具
* es6 + eslint    eslint:es6代码风格检查工具
* 工程化
* 组件化
* 模块化
* 移动端常用开发技巧
  * flex弹性布局
  * css   stickyfooter
  * 酷炫的交互设计
